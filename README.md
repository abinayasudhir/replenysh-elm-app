# Replenysh Elm App

Steps to run the app

## Installing Elm & create-elm-app packages using yarn

```sh
yarn global add elm create-elm-app
```

## Running the app

after cloning the repo

```sh
cd replenysh-elm-app
elm-app start
```

open http://localhost:3000/ to see the app in action.
